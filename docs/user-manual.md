# User Manual

This document explains how to use **Chrome Extension Template — Manifest V3** from an end-user perspective.

## Purpose

**Chrome Extension Template — Manifest V3** provides browser functionality directly inside Chrome.  
All processing happens locally in the browser.

## Installation

Users typically install the extension via:

- Chrome Web Store (once published)
- Or manual installation in developer mode (see installation guide)

## Basic Usage

After installation:

1. The extension icon appears in the browser toolbar
2. Click the icon to open the extension interface (if applicable)
3. Interact with the available features

## Data & Privacy

- No data is sent to external servers
- No personal data is collected by default
- All behavior is visible and inspectable

For details, see the privacy documentation if provided.

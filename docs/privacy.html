Chrome Extension Template — Manifest V3 – Privacy Policy

Last updated: 0.0.0

Chrome Extension Template — Manifest V3 is a client-side browser extension. All functionality runs locally in the user’s browser.

Data collection and usage

  • No personal data is collected.
  • No data is transmitted to external servers.
  • No data is sold, shared, or used for profiling.
  • All processing happens locally on the user’s device.
  • Any data stored by the extension remains local to the browser.

Permissions

Chrome Extension Template — Manifest V3 only uses the minimum browser permissions required to provide its functionality. Permissions are used solely for local processing and user-initiated actions.

Tracking and analytics

  • No analytics
  • No advertising
  • No tracking
  • No remote code execution

Third-party services

Chrome Extension Template — Manifest V3 does not integrate any third-party tracking, analytics, advertising, or data processing services.

Support

This extension is provided “as is”. Support is offered on a best-effort basis via the project’s GitHub repository.

This privacy policy applies only to Chrome Extension Template — Manifest V3. It does not cover websites, services, or platforms the extension may interact with.

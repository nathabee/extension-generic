**Manifest V3 Chrome extension template. This is the project description for Publishing.**

This project is a Chrome Extension built on **Manifest V3**.  
It is designed as a clean, minimal starting point for building browser extensions with a modern structure and clear separation of concerns.

- Project code: `extension-generic`
- Version: `0.0.0`
- Homepage: https://nathabee.github.io/extension-generic/
- Support: https://github.com/nathabee/extension-generic/issues

## Scope

This template focuses on:

- A correct Manifest V3 setup
- A predictable project structure
- A clear initialization and customization workflow
- Client-side only execution (no backend required)

It intentionally avoids opinionated frameworks or unnecessary abstractions.
